# 🚀 Sunny Raj Portfolio

A modern and fully responsive developer portfolio built using React.js, TypeScript, Tailwind CSS, and Neo-Brutalism inspired UI design.

This portfolio showcases my experience, projects, technical skills, education, achievements, certifications, and interactive terminal-based features.

---

## 🌐 Live Demo

🔗 Portfolio Live: https://new-portfolio-pi-liard.vercel.app/

---

## ✨ Features

- 🎨 Modern Neo-Brutalism UI Design
- 📱 Fully Responsive for Mobile, Tablet & Desktop
- ⚡ Interactive Terminal Component
- 🧠 Custom Blackboard Drawing Section
- 💼 Experience Timeline
- 🎓 Education Section
- 🚀 Project Showcase with GitHub & Live Links
- 📜 Certifications & Coding Achievements
- 🌙 Smooth Animations & Transitions
- 📩 Contact Modal
- 📄 Resume Download Feature

## 🚀 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/rajSunny5714/your-repo-name.git
```

Move to project folder:

```bash
cd your-repo-name
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```
